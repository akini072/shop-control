<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

2. Clone the repo
   ```sh
   git clone https://github.com/biancanfc/shop-control-client.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your BASE_URL in `.env`
   ```js
   BASE_URL=API_KEY
   ```
